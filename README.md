## ğŸ“¸ Example
### Java Input
```java
int water = 200;
int sugar = 50;
String flavor = "lime";
boolean carbonated = true;
boil(water);
add(sugar);
addFlavor(flavor);
if (carbonated) {
    mix();
}
serve();
```

### Recipe Output
```yaml

Prepare 50 g sugar  
Select "lime" as flavor

Heat 200 ml  
Add 50 g  
Add lime  
If carbonated, then:  
Mix everything  
Serve everything
```
